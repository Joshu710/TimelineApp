## Development steps

_Note: docker must be installed_

### Recommended VSCode Extensions

- Docker
- Python
- Vue
- Dev Containers

### Start development containers (within main directory)

`docker compose up --build -d`

### Restart Backend/Frontend container

`docker compose restart <backend or frontend>`

### Stop containers

`docker compose down`
